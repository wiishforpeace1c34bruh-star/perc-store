why perc

Engineered for investigators

Purpose-built architecture. Every design decision is made with speed, reliability, and operational security in mind.

::
Async Scanning
Concurrent platform checks with aiohttp and asyncio semaphores. Scan 31 platforms in under 5 seconds instead of 30+ sequential requests. Built-in concurrency limits prevent rate-limit bans.
~/
Web Dashboard
Full Flask-powered interface with session auth, search history, favorites, and stats. Run investigations from any browser.
{}
JSON Export
Every scan exports clean JSON for integration with other tools, databases, or custom pipelines. Use the -o flag to save results.
++
Modular Architecture
Each OSINT module is independent and extensible. Add custom modules, fork existing ones, or integrate with your existing toolchain. Standard SearchResult dataclass ensures consistency across all modules.
**
Secure by Default
PBKDF2 password hashing, WAL-mode SQLite, session-based auth. No telemetry, no tracking, no external dependencies at runtime.
